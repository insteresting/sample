WisOne 云开发与云协作2013-08-02

WisOne，智云集成开发平台，是一款基于云端的互联网开发平台。WisOne 向个人开发者、软件作坊以及 IT 团队提供了一个云端集成开发环境，无论何时何地，无论白天黑夜，只要灵感涌现，WisOne 可以让 IT 人员随时随地编写代码，随时随地分享创新思维。

软件开发的创新模式

随着云计算技术的发展，目前不乏各种各样基于云端的应用产品，这些产品为很多行业带来了全新的变革。而对于软件开发者来说，云端技术的灵活运用也可以让开发工作轻松自如，WisOne 提供了创新型的开发模式，开发人员只需要打开浏览器，定位到 WisOne 的网站，就可以开始编写代码，不需要安装任何额外的开发环境。更重要的是，WisOne 提供了一个社交编程平台，它进一步细化了代码分享的粒度，开发人员可以从 WisOne 社交共享环境找到自己所需要的表单、一段代码抑或是一幅精彩的图片，开发人员也可以轻松的将自己的编程过程与他人进行分享。类似于多数的协作式的托管平台一样，WisOne 也具有自己的代码托管环境，这使得专业的开发团队可以在 WisOne 进行协作开发，代码托管。

目前，软件开发行业的技术力量过于分散，项目管理也比较混乱，中国不缺优秀的软件人才，但是个人力量或者是某个小团体的力量仍然有限，借助于 WisOne，软件开发者不仅实现协作编程，并开放共享。它与多数开源项目管理不同的是，WisOne 统一了代码开发方式，它基于云端平台，提供了面向对象的 TypeScript 开发语言，利用 TypeScript 的类型化特性，使得云端的代码更容易架构和组织，杜绝低质量代码。

WisOne 的目标是云集天下编程智慧，实现软件创富梦想。在这里，将会聚集优秀的开发者、IT 团队以及 IT 企业，汇聚国内外的开源项目，开发人员可以共享 WisOne 中数以万计的开源项目，通过 WisOne 的集成打包功能，轻松实现自己的个性化软件，也可以将项目转包的方式，交由能力强劲的开发者来完成，基于社会化编程平台的项目协作模式，WisOne 可以统一开源世界无数优秀的智慧结晶，让软件创富不再是梦想。

技术构架及安全构架

WisOne 实现了一整套基于 JavaScript 的开发环境，它将 JavaScript 代码进行了面向对象的封装，对外提供TypeScript语言进行编程，TypeScript 是微软公司 C# 创始人 Heilsberg 的最新力作，TypeScript 是一种类似 C# 语法结构的强类型 Web 开发语言，它基于 JavaScript，但提供了类型检查、类、对象、接口、封装、泛型等 JavaScript 暂时缺乏的特性，这使得在 Web 上架构大型复杂的应用成为了可能。

WisOne 提供了基于事件的可视化编程环境，大大提供了软件生产的效率。当需要编写基于云端的Web应用时，只需要用鼠标拖放一些 WisOne 封装好的控件，然后对这些控件进行 TypeScript 代码编写，就可以快速的实现一些功能模块，更重要的是 WisOne 提供了数据绑定、可视化的 ORM 设计工具等，解决了目前 Web 应用开发的诸多不便，带来一种全新的 Web 应用开发体验。

在安全性方面，搭建于阿里云平台的 WisOne 系统从基础平台安全（物理安全，网络安全，系统安全）、中间应用安全（口令安全，数据安全，数据安全）、运营管理安全三个层面来设计系统安全架构，并采用阿里云数据多副本备份机制、阿里云数据快照机制以及每晚 WisOne 平台数据备份机制来保证数据安全，采用 SSH/HTTPS 保证数据传输的安全，力求给用户提供安全放心的使用环境。WisOne 系统还针对云平台环境做了适配，现有构架可以保证在增加服务器数量后，业务量可线性增长。
