随时编程 轻松共享

WisOne 可以让开发者在任何时间、任何地点编辑同样的代码，打开浏览器就可以继续编辑，无需安装，无需等待。

可视化的界面设计、可视化数据库建模等先进特性，让编程轻松有趣，不再枯燥。基于云端的 IDE 可以与世界上任何地方的开发人员分享技巧和经验，增进团队协作氛围。

WisOne 是什么？

WisOne 是一个基于 JavaScript 和 Node.js 的应用程序开发平台，具有强大的可视化编程特色，它使用 TypeScript 作为开发语言，提供了类型检查、调试、编译等特性支持。

谁会使用 WisOne？

作为一名应用程序开发者，如果您工作地点不固定，不想频繁安装和切换开发环境，或者是需要一款可视化的 Web 开发环境，WisOne 是您明智的选择。

为什么使用 WisOne？

即时开发和分享，可以让您随时随地将想法变成作品；高效的可视化开发环境，可以大大提升 Web 开发的生产效率；可视化的数据建模工具，可以让产品跨平台、跨数据库；协作和分享功能，让您远离信息技术孤岛。

无缝整合 Node.js

WisOne 无缝集成 Node.js，可以创建任何强大的服务器程序，Web 客户端程序，IDE 提供了 Node.js 的代码自动完成，实时错误信息提示并且可以运行和调试 Node.js 应用程序。

使用 TypeScript，强类型的 JavaScript 语言

WisOne 使用了 TypeScript 作为编程语言，它支持类型检查、泛型、接口、类、继承、多态等 JavaScript 难以实现的功能，使得在 WisOne 中构建大型可扩展的应用程序不再是难题。
